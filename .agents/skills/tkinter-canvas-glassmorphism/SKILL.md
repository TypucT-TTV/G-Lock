# Навык построения интерфейса на Tkinter Canvas (Glassmorphism) в G-Lock

Этот навык содержит правила и приемы создания скругленных форм кнопок, реализации эффекта полупрозрачного стекла и обеспечения адаптивности под разное системное масштабирование (DPI).

---

## 🎨 Эффект стекла (Glassmorphism) в Tkinter

Так как операционная система Windows и библиотека Tkinter не поддерживают нативное альфа-смешивание (прозрачность) для перекрывающихся виджетов (например, `ttk.Button` поверх фонового рисунка холста), в проекте G-Lock применяется специальный подход:

1. **Единое полотно (Shared Canvas)**:
   - Вся разметка главного окна рисуется как графические примитивы на одном общем холсте `tk.Canvas`.
   - Вместо встраивания отдельных виджетов, кнопки (`CanvasButton`) рисуются методами Canvas (`create_polygon`, `create_text`, `create_oval`) непосредственно в координатной сетке этого холста.
2. **Имитация прозрачности через штриховку (Stippling)**:
   - При вызове `canvas.create_polygon` передается параметр `stipple="gray50"`. Это указывает Tkinter отрисовывать заливку полигона через пиксельную маску шахматного порядка (50% прозрачности).
   - Это смешивает цвет заливки кнопки с элементами геометрического фона, находящимися позади нее на холсте.

```python
# Пример рисования полупрозрачной кнопки на Canvas
points = theme.rounded_rect_points(x, y, x + w, y + h, radius=9)
canvas.create_polygon(
    points, smooth=True, fill=theme.PANEL, outline=theme.NEON_CYAN_DIM,
    width=1.6, stipple="gray50", tags=tag
)
```

## 📏 DPI-масштабирование (DPI-Awareness)

Для предотвращения размытости интерфейса на экранах с высоким разрешением (4K, экраны ноутбуков) приложение объявляет себя DPI-aware с помощью Windows API (см. [dpi.py](file:///g:/g_lock-3.5.0/g_lock/gui/dpi.py)). 

Это отключает стандартное растягивание окна Windows, поэтому разработчик должен масштабировать все размеры вручную:

* **Масштабирующий коэффициент**: Все геометрические параметры должны умножаться на `dpi.get_scale_factor()`.
* **Правило пересчета**:
  ```python
  from gui import dpi

  scale = dpi.get_scale_factor()
  button_width = round(_BUTTON_WIDTH_BASE * scale)
  button_height = round(_BUTTON_HEIGHT_BASE * scale)
  ```

## 🛠️ Скругление прямоугольников (Smooth Polygons)

У Tkinter Canvas нет встроенного примитива `create_rounded_rectangle`. Скругление реализуется через аппроксимацию:

1. Метод `rounded_rect_points(x1, y1, x2, y2, radius)` генерирует список из 12 управляющих точек, описывающих углы и стороны.
2. Метод `canvas.create_polygon(points, smooth=True)` строит квадратичный сплайн Безье по этим точкам, превращая острые углы в гладкие закругления.

При модификации кнопок или панелей всегда используйте эту комбинацию для сохранения единого визуального стиля.
