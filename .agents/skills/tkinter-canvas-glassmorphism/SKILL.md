# Навык построения интерфейса на Svelte + CSS (Glassmorphism) в G-Lock v2

Этот навык содержит правила и приемы создания скругленных форм, реализации эффекта полупрозрачного стекла и обеспечения адаптивности под разное системное масштабирование и зум.

## 🎨 Эффект стекла (Glassmorphism) в CSS3

В веб-интерфейсе G-Lock v2 эффект стекла реализуется через современные свойства CSS3:

* **Размытие заднего плана (Backdrop Blur)**:
  `backdrop-filter: blur(20px);` в сочетании с полупрозрачным фоном `background: rgba(22, 31, 50, 0.7);` и тонкой рамкой создает премиальный эффект матового стекла.
* **Пример панели**:
  ```css
  .glass-panel {
    background: rgba(22, 31, 50, 0.75);
    backdrop-filter: blur(20px);
    border: 1px solid rgba(255, 255, 255, 0.08);
    border-radius: 12px;
    box-shadow: 0 8px 32px 0 rgba(0, 0, 0, 0.37);
  }
  ```

## 📏 CSS-масштабирование и Zoom

Масштабирование интерфейса в Svelte реализуется через CSS-трансформацию `scale` на корневом контейнере `.app-layout`.

* **Адаптивный расчет размеров**:
  Чтобы при изменении зума контейнер полностью заполнял окно без появления черных полос по бокам или снизу, его ширина и высота пересчитываются по формуле `100 / zoomFactor`:
  ```html
  <div class="app-layout" style="transform: scale({zoomFactor}); width: {100 / zoomFactor}vw; height: {100 / zoomFactor}vh;">
  ```
* **Сохранение пропорций**:
  `transform-origin: top left;` гарантирует, что уменьшение или увеличение разметки начнется из левого верхнего угла, а абсолютное позиционирование не даст дочерним элементам съехать.

## 🔑 Управление реактивным состоянием (Svelte 5)

Используйте реактивные руны Svelte 5 для связывания настроек и управления UI:
- `$state(init_value)` — для объявления реактивных переменных.
- `$derived(expression)` — для вычисляемых значений (например, `activeLang` в зависимости от `settings.language`).
- `$effect(() => { ... })` — для автоматического отслеживания зависимостей и вызова побочных эффектов (например, сохранения громкости звука в `localStorage`).
