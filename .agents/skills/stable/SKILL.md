# Воркфлоу автоматизации стабильного релиза G-Lock v2 (/stable)

Этот навык автоматизирует выпуск новой стабильной версии приложения G-Lock v2, включая обновление версионных файлов, коммит в Git и отправку тега версии на GitHub для запуска Actions-автосборки.

---

## 💡 Алгоритм выполнения воркфлоу

Когда пользователь вызывает команду `/stable`, выполните следующие шаги:

### Шаг 1: Определение версии для релиза
- Прочитайте текущую версию приложения в [package.json](file:///g:/guardian-3.5.0/package.json).
- Если версия требует инкремента, обновите её (по умолчанию инкрементируется PATCH-версия, например, `2.0.8` -> `2.0.9`).

### Шаг 2: Синхронизация версии во всех файлах проекта
Убедитесь, что одна и та же версия прописана в:
1. `package.json`
2. `src-tauri/Cargo.toml`
3. `src-tauri/tauri.conf.json`
4. `src/routes/+page.svelte` (поиск тега `<span class="ver">vX.Y.Z</span>`)

### Шаг 3: Локальная сборка (Тест)
- Запустите команду сборки для проверки работоспособности перед отправкой в репозиторий:
  ```powershell
  npm run tauri build
  ```
- Убедитесь, что сборка завершилась успешно (`exit code 0`).

### Шаг 4: Коммит изменений в Git
- Добавьте в индекс (stage) измененные файлы проекта:
  - `package.json`
  - `package-lock.json` (если изменился)
  - `src-tauri/Cargo.toml`
  - `src-tauri/Cargo.lock` (если изменился)
  - `src-tauri/tauri.conf.json`
  - `src/routes/+page.svelte`
- Закоммитьте изменения:
  ```bash
  git commit -m "release: bump version to vX.Y.Z"
  ```
- Отправьте коммит на удаленный сервер:
  ```bash
  git push origin main
  ```

### Шаг 5: Создание и отправка тега релиза
- Создайте локальный тег с именем версии (например, `v2.0.9`):
  ```bash
  git tag vX.Y.Z
  ```
- Отправьте тег на GitHub для запуска пайплайна Actions:
  ```bash
  git push origin vX.Y.Z
  ```

## 🎯 Результат работы

После завершения выведите отчет:
1. Текущая установленная стабильная версия приложения (например, `2.0.9`).
2. Подтверждение успешной отправки тега на GitHub.
3. Ссылка на вкладку Actions репозитория для отслеживания сборки релизов.
