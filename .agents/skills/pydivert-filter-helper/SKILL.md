# Навык работы с WinDivert (pydivert) в G-Lock

Этот навык предназначен для написания корректных правил фильтрации, перехвата и анализа UDP пакетов в GTA Online.

## 📌 Базовый фильтр WinDivert

Для захвата P2P-трафика GTA Online используется следующий фильтр-запрос:
```python
udp.DstPort == 6672 and udp.PayloadLength > 0 and ip
```
* **Порт 6672**: Стандартный порт Rockstar для обмена данными между игроками по протоколу P2P.
* **PayloadLength > 0**: Исключает пустые пакеты без полезной нагрузки.
* **ip**: Захватывает только IPv4 пакеты.

---

## 🛠️ Основные паттерны использования pydivert

При работе с библиотекой `pydivert` следуйте этим шаблонам:

### 1. Открытие WinDivert сессии
```python
import pydivert

# Регистрация драйвера (если не зарегистрирован)
if not pydivert.WinDivert.is_registered():
    pydivert.WinDivert.register()

# Создание сессии с фильтром и приоритетом
# Важно: приоритет (priority) определяет порядок обработки пакетов разными фильтрами
with pydivert.WinDivert(filter="udp.DstPort == 6672 and ip", priority=0) as w:
    for packet in w:
        # Обработка пакета
        ...
```

### 2. Модификация и реинжект пакетов
Каждый перехваченный пакет должен быть либо отправлен дальше (`w.send(packet)`), либо проигнорирован (отброшен):
```python
for packet in w:
    if should_allow_packet(packet):
        w.send(packet)  # Пропустить дальше в сетевой стек Windows
    else:
        # Пакет отбрасывается автоматически, если мы не вызвали w.send()
        pass
```

## 🔍 Эвристика по размеру полезной нагрузки (Payload Size)

Так как игровой трафик шифруется, тип пакета определяется по длине его полезной нагрузки:

* **Heartbeat (Пульс)**: Размеры `12`, `18`, `63` байт.
  * *Правило*: Должны пропускаться **всегда** для поддержания сессии. Если их заблокировать, игрока выбросит в оффлайн или оборвется связь со всеми.
* **Matchmaking (Попытка подключения)**: Размеры `191`, `207`, `223`, `239` байт.
  * *Правило*: Блокировка этих размеров предотвращает появление новых игроков в сессии (режим Locked).
* **Служебные обертки (Rockstar Tunnel)**: Контрольные байты по смещениям (XOR-паттерны) для более сложного анализа сессий по вайтлисту.

## ⚠️ Правила безопасности и утечки ресурсов

1. **Гарантированное закрытие WinDivert**: Всегда оборачивайте сессию `pydivert.WinDivert` в менеджер контекста `with` или вызывайте `.close()` в блоке `finally`. Незакрытый дескриптор WinDivert заблокирует трафик на порту 6672 даже после закрытия программы.
2. **Многопроцессность**: Чтение пакетов из WinDivert в цикле — блокирующая операция. Код должен запускаться в дочернем процессе. Для завершения процесса используйте метод `process.terminate()` и последующий `process.join()`.
