====================================================
 G-LOCK — ФАЕРВОЛ ДЛЯ GTA ONLINE. ИНСТРУКЦИЯ (RU)
====================================================

ЧТО ЭТО ТАКОЕ
--------------
G-Lock — это локальный сетевой фаервол, который помогает управлять тем,
кто может подключиться к вашей сессии GTA Online (защита от рандомов,
модеров и нежелательных подключений). Он не читает и не изменяет память
игры — работает только на уровне сетевых пакетов (через драйвер WinDivert).

ЗАПУСК
------
При запуске G-Lock автоматически запросит права Администратора (через UAC) —
это необходимо для регистрации драйвера WinDivert и перехвата сетевых пакетов.

ГЛАВНОЕ ОКНО
------------
Вверху — цветной индикатор и текстовый статус:
  * Зелёный / "ОТКРЫТО"        — сессия открыта для всех, кроме игроков из черного списка.
  * Красный / "ЗАПЕРТО"        — сессия заблокирована для новых игроков (в фоне блокируются новые входящие P2P-пакеты).

Иконка приложения в панели задач Windows также динамически меняет цвет:
  * Зеленый круг — режим "Открыто" (безопасно, стандартная фильтрация).
  * Красный круг — режим "Заперто" (вход заблокирован).

КНОПКА "ОТКРЫТЬ СЕССИЮ"
-------------------------
Сбрасывает текущий активный режим фильтрации и возвращает фаервол в режим по умолчанию — "Открыто" (с фоновой фильтрацией черного списка).

КНОПКИ РЕЖИМОВ СЕССИИ
-----------------------
Каждая кнопка активирует или деактивирует (при повторном клике) соответствующий режим:

1. Запереть сессию (Lock)
   Блокирует новые попытки игроков зайти в ваше лобби, но не трогает тех, кто уже находится внутри.
   * Этот режим можно быстро включать/выключать клавишей F9 в игре.
   * Пока сессия открыта, G-Lock запоминает IP уже подключенных участников. В Lock они продолжают работать, а неизвестный non-service трафик блокируется.
   * ВНИМАНИЕ: Перед сменой сессии сначала разблокируйте её (F9, Ctrl+F9 или "Открыть сессию"), найдите новое лобби и только после загрузки заприте обратно.

2. Solo-сессия
   Полностью блокирует входящие соединения от других игроков. Вся текущая сессия разделится, и вы останетесь в лобби абсолютно один. При этом критически важный трафик Rockstar и Social Club продолжает проходить.

3. Сессия по вайтлисту
   В открытом состоянии служебный handshake Rockstar проходит, но прямой P2P-трафик разрешается только IP-адресам и подсетям из Вайтлиста. Реальный IP неизвестного игрока блокируется в обоих направлениях и появляется в журнале.
   В закрытом состоянии новые игроки не допускаются вообще, включая новые IP из Вайтлиста; уже известные участники продолжают работать как в обычном Lock.
   Черный список имеет абсолютный приоритет во всех состояниях и блокирует адрес в обоих направлениях.

ГОРЯЧАЯ КЛАВИША F9
------------------
Клавиша F9 переключает состояние "Открыто/Заперто". В режиме Вайтлиста она сохраняет активный режим: открыто допускает только Вайтлист, заперто не допускает новых игроков. В остальных режимах сохраняется обычное поведение Open/Lock.
  * Работает глобально, даже когда окно игры GTA находится в фокусе и развернуто на весь экран.
  * Первое нажатие F9 — запирает сессию.
  * Повторное нажатие F9 — отпирает сессию.
  * Также доступно экстренное разблокирование по сочетанию Ctrl + F9.

МАСШТАБИРОВАНИЕ ИНТЕРФЕЙСА
---------------------------
Для удобства использования на экранах с высоким разрешением вы можете изменять масштаб UI:
  * Ctrl + (или Ctrl =) — увеличить масштаб интерфейса.
  * Ctrl - — уменьшить масштаб интерфейса.
  * Ctrl 0 — сбросить масштаб к стандартным 100%.
  Текущий масштаб автоматически сохраняется и применяется при следующем запуске.

АДАПТИВНЫЙ ФЛУД-ДЕТЕКТОР (IPS)
------------------------------
G-Lock оснащен интеллектуальной системой предотвращения вторжений (IPS):
  * При старте фильтрации программа замеряет нормальную (фоновую) интенсивность трафика от других игроков (по умолчанию 45 секунд).
  * Рассчитывается динамический лимит пакетов в секунду (PPS) с учетом множителя. Если неизвестный IP-адрес начинает флудить пакетами выше лимита, он временно блокируется на время бана (по умолчанию 60 секунд).
  * Релей-серверы Rockstar/Azure полностью исключены из системы IPS-банов во избежание сбоев авторизации.
  * Дополнительный глобальный PPS-предел обнаруживает распределенный флуд со множества меняющихся IP. Внутренние таблицы ограничены по размеру и очищаются по времени жизни записей.

ЛОГ ПОДКЛЮЧЕНИЙ И ИНТЕРАКТИВНАЯ ПАНЕЛЬ
---------------------------------------
G-Lock отображает все сетевые подключения к вам в реальном времени.

Быстрые кнопки в начале каждой строки:
  * `🟢` (WL) — мгновенно добавить IP-адрес игрока в Белый список.
  * `🔴` (BL) — мгновенно добавить IP-адрес игрока в Черный список.

Цветовая разметка лога:
  * Жёлтый  — друзья из вашего белого списка (Вайтлист).
  * Зелёный — обычные разрешенные P2P-подключения других игроков.
  * Синий   — служебные транзитные подключения через серверы Rockstar (`[RELAY R*]`).
  * Красный — заблокированные попытки подключения.

СПИСКИ (ВАЙТЛИСТ / БЛЭКЛИСТ)
-----------------------------
Вкладка "Списки защиты" позволяет управлять вашими списками адресов:
  * Вы можете вручную добавлять, редактировать или удалять IP-адреса и подсети в формате CIDR (например, `192.168.1.0/24`).
  * Кнопка "Очистить" позволяет полностью сбросить список.
  * Любое изменение списка мгновенно вступает в силу без перезапуска фаервола.
  * При добавлении адреса в один список он автоматически удаляется из противоположного. Принимаются только корректные IPv4-адреса и IPv4 CIDR.

БЕЗОПАСНОСТЬ
------------
G-Lock не добавляет свою папку в исключения Windows Defender. Если антивирус блокирует подписанные файлы WinDivert, проверьте предупреждение и при необходимости создайте вручную максимально узкое исключение.
