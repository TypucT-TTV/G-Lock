====================================================
 G-LOCK — ФАЕРВОЛ ДЛЯ GTA ONLINE. ИНСТРУКЦИЯ (RU)
====================================================

ЧТО ЭТО ТАКОЕ
--------------
G-Lock — это локальный сетевой фаервол, который помогает управлять тем,
кто может подключиться к вашей сессии GTA Online (защита от рандомов,
модеров и нежелательных подключений). Он не читает и не изменяет память
игры — работает только на уровне сетевых пакетов (через драйвер WinDivert).

ЗАПУСК
------
При запуске G-Lock автоматически запросит права Администратора (через UAC) —
это необходимо для регистрации драйвера WinDivert и перехвата сетевых пакетов.

ГЛАВНОЕ ОКНО
------------
Вверху — цветной индикатор и текстовый статус:
  * Зелёный / "ОТКРЫТО"        — новые peer-подключения разрешены; IPS и ручные блокировки IP продолжают работать.
  * Красный / "ЗАПЕРТО"        — сессия заблокирована для новых игроков (в фоне блокируются новые входящие P2P-пакеты).

Иконка приложения в панели задач Windows также динамически меняет цвет:
  * Зеленый круг — режим "Открыто" (безопасно, стандартная фильтрация).
  * Красный круг — режим "Заперто" (вход заблокирован).

УПРАВЛЕНИЕ СЕССИЕЙ
------------------
У G-Lock одно управление: "Открыто/Заперто". Кнопка находится прямо в карточке статуса.
  * "Заперто" блокирует новые неизвестные peer-подключения, но сохраняет участников, которых G-Lock уже обнаружил во время текущего запуска.
  * "Открыто" снова разрешает новые подключения. G-Lock при этом не выключается: IPS и ручные блокировки IP продолжают работать.
  * Перед сменой лобби откройте сессию, дождитесь загрузки и только после этого заприте её снова.

ПОЧЕМУ НЕТ SOLO И WHITELIST
---------------------------
Входящий игрок сначала общается через инфраструктуру Rockstar, а полезный прямой peer IP становится виден позже. Эвристики по размеру UDP-пакета не позволяют гарантированно отличить весь служебный трафик от подключения игрока. Поэтому режимы Solo и Whitelist убраны из поддерживаемого интерфейса и не выдаются за надёжную защиту.

ГОРЯЧАЯ КЛАВИША F9
------------------
Клавиша F9 переключает состояние "Открыто/Заперто".
  * Работает глобально, даже когда окно игры GTA находится в фокусе и развернуто на весь экран.
  * Первое нажатие F9 — запирает сессию.
  * Повторное нажатие F9 — отпирает сессию.
  * Также доступно экстренное разблокирование по сочетанию Ctrl + F9.

МАСШТАБИРОВАНИЕ ИНТЕРФЕЙСА
---------------------------
Для удобства использования на экранах с высоким разрешением вы можете изменять масштаб UI:
  * Ctrl + (или Ctrl =) — увеличить масштаб интерфейса.
  * Ctrl - — уменьшить масштаб интерфейса.
  * Ctrl 0 — сбросить масштаб к стандартным 100%.
  Текущий масштаб автоматически сохраняется и применяется при следующем запуске.

АДАПТИВНЫЙ ФЛУД-ДЕТЕКТОР (IPS)
------------------------------
G-Lock оснащен интеллектуальной системой предотвращения вторжений (IPS):
  * При старте фильтрации программа замеряет нормальную (фоновую) интенсивность трафика от других игроков (по умолчанию 45 секунд).
  * Рассчитывается динамический лимит пакетов в секунду (PPS) с учетом множителя. Если неизвестный IP-адрес начинает флудить пакетами выше лимита, он временно блокируется на время бана (по умолчанию 60 секунд).
  * Релей-серверы Rockstar/Azure полностью исключены из системы IPS-банов во избежание сбоев авторизации.
  * Дополнительный глобальный PPS-предел обнаруживает распределенный флуд со множества меняющихся IP. Внутренние таблицы ограничены по размеру и очищаются по времени жизни записей.

ЛОГ ПОДКЛЮЧЕНИЙ И ИНТЕРАКТИВНАЯ ПАНЕЛЬ
---------------------------------------
G-Lock отображает решения по перехваченным GTA P2P-пакетам в реальном времени. Это не подтверждённый список игроков.

Relay-адрес Rockstar/Azure может появиться раньше прямого соединения. Реальный peer IP становится виден только после начала прямого P2P-трафика. Relay IP не является идентификатором игрока, G-Lock не умеет сопоставлять IP с ником GTA, а несколько игроков иногда используют один внешний адрес через NAT/CGNAT.

Клик по IP копирует адрес. Быстрых кнопок мгновенной блокировки в журнале нет, чтобы снизить риск блокировки relay или общего адреса NAT/CGNAT.

Цветовая разметка лога:
  * Зелёный — обычные разрешенные P2P-подключения других игроков.
  * Синий   — служебные транзитные подключения через серверы Rockstar (`[RELAY R*]`).
  * Красный — заблокированные попытки подключения.

ДОПОЛНИТЕЛЬНЫЕ БЛОКИРОВКИ IP
----------------------------
Раздел "Настройки → Дополнительно: заблокированные IP" принимает IPv4 и CIDR (например, `192.168.1.0/24`). Правило мгновенно отбрасывает захваченный трафик адреса в обоих направлениях после появления прямого peer IP.

Это не бан игрока по нику и не гарантия предотвращения первоначального входа через Rockstar. Внешние IP меняются, а один адрес может использоваться несколькими игроками через NAT/CGNAT. Добавляйте только адреса, которые проверили независимо.

БЕЗОПАСНОСТЬ
------------
G-Lock не добавляет свою папку в исключения Windows Defender. Если антивирус блокирует подписанные файлы WinDivert, проверьте предупреждение и при необходимости создайте вручную максимально узкое исключение.

ГРАНИЦЫ РАБОТЫ
---------------
G-Lock фильтрует IPv4 UDP P2P-трафик GTA на порту 6672. Он не внедряет код, не читает память игры, не определяет игроков по нику и не анализирует все соединения Windows. IPS ограничивает подозрительный перехваченный трафик, но не гарантирует защиту всего интернет-канала от любой объёмной DDoS-атаки. В логах и data.json могут находиться внешние IP — проверяйте файлы перед публикацией.
