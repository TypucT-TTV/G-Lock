====================================================
 G-LOCK — GTA ONLINE FIREWALL. INSTRUCTIONS (EN)
====================================================

WHAT THIS IS
------------
G-Lock is a local network firewall that helps control who can connect to
your GTA Online session (protection from randoms, modders, and unwanted
connections). It does not read or modify the game's memory — it only
operates at the network packet level (via the WinDivert driver).

LAUNCHING
---------
When launched, G-Lock will automatically request Administrator privileges (via UAC).
This is required to register the WinDivert driver and capture network packets.

MAIN WINDOW
-----------
At the top: a colored indicator and a text status:
  * Green / "OPEN"            — new peer connections are allowed; IPS and manual IP blocks remain active.
  * Red / "LOCKED"            — new player joins are blocked (background filter drops incoming P2P packets).

The app's taskbar icon also changes color dynamically:
  * Green circle — "Open" mode (safe, default background filtering active).
  * Red circle   — "Locked" mode (new join attempts are blocked).

The version state appears below the logo. "Up to date" means the installed version
matches the latest published release. "Update available" opens the GitHub download
for the new installer. If the network is unavailable, G-Lock continues to work and
lets you retry the check by clicking the message.

SESSION CONTROL
---------------
G-Lock has one control: Open/Locked. The button is located directly in the status card.
  * Locked rejects unknown new peer connections but keeps peers that G-Lock already detected during the current run.
  * Open allows new connections again. It does not turn G-Lock off: IPS and manual blocked-IP rules remain active.
  * Before changing lobby, open the session, wait for it to load, and then lock it again.

WHY SOLO AND WHITELIST ARE NOT OFFERED
--------------------------------------
A joining player first communicates through Rockstar infrastructure, while the useful direct peer IP appears later. UDP packet-size heuristics cannot reliably distinguish every service packet from a player connection. Solo and Whitelist were therefore removed from the supported UI instead of being presented as dependable protection.

GLOBAL HOTKEY
-------------
F9 toggles Open/Locked by default. Under "Settings → Global hotkey", click the
capture field and press an F-key or a combination containing Ctrl or Alt with a letter,
number, numpad key, or navigation key; Shift may be added. Non-F keys without Ctrl/Alt are rejected so a
global shortcut cannot interfere with normal typing. The shortcut applies after
clicking "Save settings".
  * Works globally even when the GTA game window is focused and running in full screen.
  * Press the selected key — locks the session.
  * Press it again — unlocks the session.
  * Ctrl + F9 is the fixed fallback open action. It forces Open but does not disable
    G-Lock, IPS, or persistent blocked-IP rules.

SETTINGS PERSISTENCE
--------------------
Settings, the selected hotkey, and blocked IP rules are stored in
`%LOCALAPPDATA%\G-Lock\data.json`. Installing an update over the existing version
does not reset them. On first launch, an older `data.json` next to the executable is
migrated automatically.

INTERFACE ZOOM CONTROL
-----------------------
You can scale the interface to fit high-resolution screens:
  * Ctrl + (or Ctrl =) — zoom in.
  * Ctrl - — zoom out.
  * Ctrl 0 — reset zoom to 100%.
  Your zoom preference is saved and applied automatically on the next launch.

ADAPTIVE FLOOD DETECTOR (IPS)
-----------------------------
G-Lock includes an intelligent Intrusion Prevention System (IPS):
  * On session start, G-Lock measures normal background traffic from peers (default: 45 seconds).
  * A dynamic Packets Per Second (PPS) threshold is calculated. If a peer exceeds this threshold, they are banned for a cooldown period (default: 60 seconds) to prevent session lags.
  * Rockstar/Azure relays are fully exempt from IPS bans to avoid cloud save/sync errors.
  * A global PPS ceiling detects distributed floods from many changing IPs. Internal tracking tables are bounded and expire old entries.

CONNECTION LOG & INTERACTIVE PANEL
----------------------------------
G-Lock shows decisions for captured GTA P2P packets in real time. It is not a verified player roster.

A Rockstar/Azure relay address can appear before the direct peer connection. The real peer IP becomes visible only when direct P2P traffic starts. A relay IP is not a player identity, G-Lock cannot map an IP to a GTA nickname, and multiple players can sometimes share one public IP through NAT/CGNAT.

Click an IP to copy it. The log intentionally has no one-click blocking buttons, reducing the risk of blocking a relay or a shared NAT/CGNAT address.

Color coding:
  * Green  — normal allowed P2P player connections.
  * Blue   — transit connections through Rockstar servers (`[RELAY R*]`).
  * Red    — blocked connection attempts.

ADVANCED BLOCKED IP RULES
-------------------------
"Settings → Advanced: Blocked IP addresses" accepts IPv4 and CIDR rules (for example, `192.168.1.0/24`). A rule immediately drops captured traffic in both directions after the direct peer IP becomes visible.

This is not a nickname-based player ban and cannot guarantee prevention of the initial Rockstar-mediated join. Public IPs change, and multiple players may share one address through NAT/CGNAT. Add only addresses you have independently verified.

SECURITY
--------
G-Lock does not add its folder to Windows Defender exclusions. If security software blocks the signed WinDivert files, review the alert and configure the narrowest possible exception manually if required.

SCOPE AND LIMITATIONS
---------------------
G-Lock filters IPv4 UDP P2P traffic on GTA's port 6672. It does not inject code, read game memory, identify players by nickname, or inspect every Windows connection. IPS rate-limits suspicious captured traffic but cannot guarantee protection of the entire internet connection from every volumetric DDoS attack. Connection logs and `%LOCALAPPDATA%\G-Lock\data.json` may contain public IP addresses; review them before sharing.
