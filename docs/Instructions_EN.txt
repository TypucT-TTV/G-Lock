====================================================
 G-LOCK — GTA ONLINE FIREWALL. INSTRUCTIONS (EN)
====================================================

WHAT THIS IS
------------
G-Lock is a local network firewall that helps control who can connect to
your GTA Online session (protection from randoms, modders, and unwanted
connections). It does not read or modify the game's memory — it only
operates at the network packet level (via the WinDivert driver).

LAUNCHING
---------
G-Lock MUST be run as Administrator — without that, the WinDivert driver
can't be registered, and the app will refuse to start (it will show a
message explaining this).

MAIN WINDOW
-----------
At the top: a colored indicator and a text status:
  * Green / "Open"            — session open for everyone except players in your blacklist (runs PrivateSession with locked=False in the background).
  * Red / "LOCKED"            — a Locked session is active (runs PrivateSession with locked=True, blocks new joins).
  * "Active: Solo"            — Solo session is currently running (SoloSession).

The app's taskbar icon also changes color (green/red) in sync with this
status.

"STOP SESSION" BUTTON
----------------------
A separate button specifically for TURNING OFF whatever mode is currently
active. Clicking it disables the active mode and returns to the default "Open" background blacklist filter state.

SESSION MODE BUTTONS (each has an indicator dot on the left)
--------------------------------------------------------------
The dot next to each button lights up green if that specific mode is the
one currently active.

1. Lock Session
   Blocks NEW join attempts, but doesn't touch anyone already inside. This
   exact same thing can be toggled with the F9 hotkey (see below).
   NOTE: While locked, matchmaking packets are blocked. To switch to another populated session, you MUST first unlock the session (F9 or "Open" in the GUI), search for a new session, and lock it again (F9) only after you successfully join. Service packets (Heartbeats) are always allowed to prevent any Social Club or cloud save connection issues.

2. Solo Session
   No one can connect to you, but critical R* and Social Club traffic still
   goes through. If you were in a session with others, they'll lose
   connection to you.

3. Empty Session
   Splits you off from the current session so you're alone — guaranteeing
   you become its host.

F9 HOTKEY
---------
F9 is a separate, standalone toggle specifically for "Locked Session". It
does NOT require pressing any other button first — it works on its own,
even while GTA has focus and is running in exclusive fullscreen.
  * Press F9              — the session locks, no new joins.
  * Press F9 again         — open to everyone again.
Note: G-Lock only ever keeps ONE filter active at a time. If another
session type was already running (e.g. Solo), F9 will REPLACE it
with Lock. Pressing F9 again returns you to "Open", not back to whatever
was running before.

CONNECTION LOG (WHO JOINED, AND WHEN)
CONNECTION LOG & INTERACTIVE PANEL
----------------------------------------
G-Lock keeps a passive log of all connections and displays it in real-time in the right panel of the main window.

The panel features intuitive color coding:
  * Green — friends joining directly from residential IPs.
  * Blue  — connections passing through Rockstar/Azure relays (typically modders, hidden IPs, or closed NAT).
  * Red   — blocked packet attempts.
  * Grey  — service/passive packets.

Quick log actions (right-click a row in the log table):
  * "Copy IP" — copy the player's IP address.
  * "Add to Whitelist" / "Add to Blacklist" — quickly add the IP to your lists with a name. The protection rules will reload automatically on the fly without restarting the program.

Logs are saved as UTF-8 files in the "logs" folder next to the app, keeping only the 10 most recent days.

EDIT LISTS (WHITELIST / BLACKLIST)
------------------------------------
The "Edit lists" button opens a window with Whitelist / Blacklist tabs.
You can manage entries manually (Add, Edit, Delete).
G-Lock supports adding both individual IPs and entire subnet ranges in CIDR format (e.g., 52.139.169.0/24), which is convenient for blocking whole Rockstar/Azure relay ranges.
Config files are stored in UTF-8, allowing names to contain Cyrillic/non-English characters.

LANGUAGE SWITCHER
------------------
At the bottom of the window: "RU" / "EN" buttons. They switch the entire
interface's language on the fly, and your choice is remembered across
restarts.

IF SOMETHING GOES WRONG
-------------------------
On any failure, the app saves a crashreport_*.log file next to itself and
shows a window describing the error — this is a deliberate safeguard
against silently vanishing without a trace, not a bug.
