====================================================
 G-LOCK — GTA ONLINE FIREWALL. INSTRUCTIONS (EN)
====================================================

WHAT THIS IS
------------
G-Lock is a local network firewall that helps control who can connect to
your GTA Online session (protection from randoms, modders, and unwanted
connections). It does not read or modify the game's memory — it only
operates at the network packet level (via the WinDivert driver).

LAUNCHING
---------
When launched, G-Lock will automatically request Administrator privileges (via UAC).
This is required to register the WinDivert driver and capture network packets.

MAIN WINDOW
-----------
At the top: a colored indicator and a text status:
  * Green / "OPEN"            — session open for everyone except players in your blacklist.
  * Red / "LOCKED"            — new player joins are blocked (background filter drops incoming P2P packets).

The app's taskbar icon also changes color dynamically:
  * Green circle — "Open" mode (safe, default background filtering active).
  * Red circle   — "Locked" mode (new join attempts are blocked).

"OPEN SESSION" BUTTON
----------------------
Disables the active filtration mode and returns the firewall to the default "Open" background blacklist filter state.

SESSION MODE BUTTONS
---------------------
Each button toggles the respective filtration mode on or off (clicking an active mode turns it off):

1. Lock Session
   Blocks new players from entering your session, but doesn't disconnect anyone already inside.
   * This mode can be toggled using the F9 hotkey inside the game.
   * While Open, G-Lock learns the IPs of active peers. Lock keeps those known peers connected and blocks unknown non-service traffic.
   * NOTE: Before joining another populated session, unlock the firewall (F9, Ctrl+F9, or "Open Session"), join the new lobby, and lock it again once loaded.

2. Solo Session
   Blocks all incoming connections from other players. Your current session will split, leaving you completely alone in the lobby. Critical Rockstar and Social Club connection traffic remains allowed.

3. Whitelist Session
   While open, Rockstar signaling is allowed but direct P2P traffic is accepted only from IP addresses and subnets in your Whitelist. An unknown player's real IP is blocked in both directions and shown in the log.
   While locked, no new player is accepted, including a newly seen whitelisted IP; already-known peers remain connected as in normal Lock mode.
   The Blacklist has absolute priority in every state and blocks an address in both directions.

F9 HOTKEY
---------
F9 toggles the Open/Locked state globally. In Whitelist mode it preserves that mode: Open admits only whitelisted peers, while Locked admits no new peers. Outside Whitelist mode it keeps the normal Open/Lock behavior.
  * Works globally even when the GTA game window is focused and running in full screen.
  * Press F9 — locks the session.
  * Press F9 again — unlocks the session.
  * Emergency panic unlock is also available via Ctrl + F9.

INTERFACE ZOOM CONTROL
-----------------------
You can scale the interface to fit high-resolution screens:
  * Ctrl + (or Ctrl =) — zoom in.
  * Ctrl - — zoom out.
  * Ctrl 0 — reset zoom to 100%.
  Your zoom preference is saved and applied automatically on the next launch.

ADAPTIVE FLOOD DETECTOR (IPS)
-----------------------------
G-Lock includes an intelligent Intrusion Prevention System (IPS):
  * On session start, G-Lock measures normal background traffic from peers (default: 45 seconds).
  * A dynamic Packets Per Second (PPS) threshold is calculated. If a peer exceeds this threshold, they are banned for a cooldown period (default: 60 seconds) to prevent session lags.
  * Rockstar/Azure relays are fully exempt from IPS bans to avoid cloud save/sync errors.
  * A global PPS ceiling detects distributed floods from many changing IPs. Internal tracking tables are bounded and expire old entries.

CONNECTION LOG & INTERACTIVE PANEL
----------------------------------
G-Lock shows connection logs in real-time.

Quick buttons next to each row:
  * `🟢` (WL) — instantly add the player's IP to the Whitelist.
  * `🔴` (BL) — instantly add the player's IP to the Blacklist.

Color coding:
  * Yellow — whitelisted friends (to see when friends join).
  * Green  — normal allowed P2P player connections.
  * Blue   — transit connections through Rockstar servers (`[RELAY R*]`).
  * Red    — blocked connection attempts.

EDIT LISTS (WHITELIST / BLACKLIST)
----------------------------------
The "Lists" tab allows you to manage whitelist and blacklist addresses:
  * You can manually add, edit, or delete IP addresses and subnets in CIDR format (e.g. `192.168.1.0/24`).
  * The "Clear" button resets the list.
  * Any list modifications take effect immediately without restarting the firewall.
  * Adding an address to one list automatically removes it from the other. Only valid IPv4 addresses and IPv4 CIDR networks are accepted.

SECURITY
--------
G-Lock does not add its folder to Windows Defender exclusions. If security software blocks the signed WinDivert files, review the alert and configure the narrowest possible exception manually if required.
